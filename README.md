## 🔧 Tecnologias usadas
- HTML5
- CSS3

## 📁 Funcionalidades
- Layout responsivo
- Estilização moderna
- Organização semântica do código

## 🧠 Objetivo
Prática de front-end com foco em estrutura de páginas e design responsivo.
